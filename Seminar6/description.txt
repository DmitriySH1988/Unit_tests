**Сценарии покрыты тестами:**

1. `test_compare_averages_first_list_bigger`: Проверяет случай, когда среднее значение первого списка больше среднего значения второго списка. 
Этот тест покрывает сценарий, когда программа должна вернуть сообщение "Первый список имеет большее среднее значение".

2. `test_compare_averages_second_list_bigger`: Проверяет случай, когда среднее значение второго списка больше среднего значения первого списка. 
Этот тест покрывает сценарий, когда программа должна вернуть сообщение "Второй список имеет большее среднее значение".

3. `test_compare_averages_equal`: Проверяет случай, когда средние значения обоих списков равны. 
Этот тест покрывает сценарий, когда программа должна вернуть сообщение "Средние значения равны".

4. `test_average_calculator_empty_list`: Проверяет, что калькулятор среднего значения корректно обрабатывает пустой список.
Этот тест покрывает сценарий, когда входной список пуст.

**Почему выбраны именно эти сценарии:**

Выбор этих сценариев обусловлен тем, что они представляют различные возможности и результаты работы программы:

- Сценарии 1 и 2 покрывают случаи, когда один из списков имеет большее среднее значение, что является ключевой функциональностью программы.
- Сценарий 3 проверяет ситуацию, когда средние значения обоих списков равны, что также важно для корректности работы программы.
- Сценарий 4 гарантирует, что калькулятор среднего значения корректно обрабатывает пустой входной список, 
  что представляет собой частный случай и важно для обеспечения корректности программы.

Таким образом, выбор этих сценариев позволяет убедиться в правильности работы программы в различных ситуациях и обеспечить ее надежность.
